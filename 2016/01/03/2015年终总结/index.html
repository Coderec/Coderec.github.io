2015年终总结

发表于 2016-01-03   |   分类于 个人总结   |     |   阅读次数

2015年结束了，年终总结！

题图： from Google

0x01 概述

2015年，结束了4年的大学
2015年，开始了人生的第一份工作
2015年，上半年还忙着高层建筑给排水设计
2015年，下半年已忙着代码上线修复bug
2015年，发生了很多事，也有了身份和心态的转变
但总的来讲，2015年，让自己变得更好

0x02 结束了的四年

大学，四年，终究还是一不留神就过去了。
我的大学，和最普通的大学生一样。不是学霸，不是学渣，因为玩游戏同时挂掉高数大物，因为兴趣通宵编程。
大学前，我不看球，我不知道什么编程，我不认识现在珍惜的这么多同学。

对于四年大学生活，我想说的只有感谢。
她给了我足够的时间浪费，给我足够的时间了解，给我足够的时间学习，最重要的，给了我足够的时间去思考。
浪费时间去堕落与空虚，抓紧时间去了解自己的兴趣与学习，以及充足的时间去思考自己想要什么。

当然，对于大学的挚友们，我想说的也是感谢。
不论是带给我的正能量，亦或是带我一起入坑游戏，再或是一起谈心或是旅行游荡，我相信，我之所以是今天的我，离不开大家的帮助与影响。

对于这结束了的四年，遗憾是有的。连200的奖学金都没拿过有木有……
但是最让我感到值得的是，我用四年的时间，找到了自己的兴趣，明确了自己的目标。
无论如何，四年时间，还是结束了。

0x03 开始了的半年

自己找房、自己找面试、找工作、自己赚钱自己花。
从大学生变为工作狗，积极的心态，多学、多做，一只菜鸟渴望成长。

工作的半年，最应该感谢的；
是同学的帮助，让我没有吃到苦，找到房子前一直在蹭住；
是上司的教导，让我明确学习的方向与方法；
是同事的宽容，容忍我犯得低级错误；

开始了的半年，成功从环工人转型为程序猿，换了自己的第一个Mac。
积极工作、努力学习、不断前进，坚持->胜利。

0x04 知识提升

  1. 深入学习了Git版本控制的原理与使用
  2. 构建管理工具Gradle
  3. Java Web开发常用框架技术：JavaMail、HttpClient、Apache Poi、Hibernate Validator、Spring Data JPA
  4. 对整洁代码以及代码重构有了更深的感悟

0x05 2016计划

  1. 学习上要按照既定计划，先广后精，将用的到的技术都进行了解并可以使用，而后在进行深入。
  2. 工作上要多思考业务场景，使代码稳定，兼容性提高。
  3. 生活上，多读书，去除码农属性；重新拿起画笔，无聊涂鸦也是一种不错的娱乐方式。

0x06 后记

2015年末，与在北京的同学聚了聚，感慨有些人变了，有些人还是老样子。

想想自己，应该也是变化了吧。从环境工程成功转型为代码农民工，从迷茫不定到坚定不移，有了奋斗的目标，日子确实是越过越好了。

2016，还要继续加油。

Mac下好用的工具

发表于 2016-04-26   |   分类于 推荐   |     |   阅读次数

来点儿APP推荐

题图：from Zoommy

引言

使用Mac也有将近一年时间了，从Windows到OS X的切换并没有想象中那么困难，直接使用，过渡得十分平滑。对于刚转到OS X系统的新手来说，除了急需找到Windows平台上的对应替换软件之外，可能最需要的就是OS X平台上用的顺手的各种生产力工具了。

因此，今天就说说我这一年使用过的好应用。

干货

程序猴子专场

  1. Go2Shell 送你去APP Store
    这是一款Finder插件，可以直接在终端中打开当前目录。
  2. dash 官网链接
    API文档大集合，囊括了N多API，简直是查文档利器。
  3. cyberduck 官网链接
    FTP、SFTP连接远程服务器的小鸭子，个人觉得比FileZilla好用。
  4. iterm 官网链接
    用过OS X自带的terminal了吗，别说跟Ubuntu的自带终端比了，怎么感觉比Windows的cmd也好不到哪去呢？
    好吧，那就去试试iterm吧，分屏、透明度、弹出方式等等特性，照着网上的教程（一把一把的）去发掘吧。
  5. tmux CaltYang的博客
    一款可以保存session、分屏的终端工具，简直就是盯日志神器，当然，他的犀利不止于此。
  6. charles 官网链接
    http代理服务器，上手简单，使用方便，挂个代理看看数据什么的，想想都激动。
  7. sequel pro 官网链接
    还记得Windows下的Navicat吗，OS X下的Mysql客户端就改用Sequel吧。
  8. parallels desktop 收费死贵的官网
    OSX上最好用的虚拟机，没有之一，Vbox什么的都是。。不提了。

有时候，我只想做一个普通人

：），除了上班当猴子，下班我也想享受OSX的流畅体验，接下来就说说平时也能用到的好应用。

  1. cleanMyMac官网链接
    OSX系统的管家软件，作为一个普通用户没有电脑管家怎么能行？
    其实Mac卸载软件只需要把app拖到废纸篓就好了，但是总有精神洁癖以及怕出任何问题的人不是？
    用CleanMyMac吧，解决你的烦恼
  2. mindnode官网链接
    思维导图，没有必要多说了吧
  3. reederApp Store
    RSS订阅，从此不再各个网站来回跑。
  4. zoommy官网链接
    收集网络上的免费高质量图片，需要图片？这款软件就够了。
  5. window tidyApp Store
    窗口排列神器，轻松改变窗口大小，停靠位置。
  6. ulyssesApp Store
    用Markdown写作吗？如果答案是肯定的，墙裂向你推荐ulysses，除了常规的编辑作用外，还有文库管理，搜索，预览，版本控制等功能，体验极佳。
  7. istat Menus官网链接
    一款在状态栏显示系统信息的插件，内存占用、cpu使用率、网速、温度，尽收眼底。
  8. unclutterApp Store
    隐藏在屏幕上方的便签工具，不仅仅能记录剪切板，还能暂存文件以及记录便签。

收工

暂时想到的应用就是这些，希望大家能找到自己喜欢的软件：）。

PS：用Mac的人怎么能不知道的网站：www.macx.cn

PPS：有能力支持正版，还是多支持正版吧：）。

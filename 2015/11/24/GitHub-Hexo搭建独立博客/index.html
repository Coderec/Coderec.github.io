GitHub+Hexo搭建独立博客

发表于 2015-11-24   |   分类于 配置记录   |     |   阅读次数

知识点，只有在能给别人讲通时，才算真正掌握了它。

为什么搞博客

还在上学的时候就搞过独立博客，当时只是觉得好玩。这次决定重新把博客建立起来，主要是为了记录自己技术的积累，为这个博客专门买了域名，也是督促自己认真对待，不要把这个博客荒废了。

Hexo+NexT

使用hexo是因为简单、快速、易用，主题选择NexT是因为简洁美观。

搭建过程

因为是在Mac上搭建的，就说说Mac上的搭建过程

  1. 注册一个『GitHub』帐号，我的账户名为Coderec
  2. 建立与你用户名对应的仓库，仓库名必须为『coderec.github.io』
  3. 添加本机的SSH公钥到『Account settings -> SSH Keys -> Add SSH Key』
  4. HomeBrew的安装
  5. brew install git
  6. brew install node
  7. npm install hexo -g –no-optional (我这里安装好的hexo版本是3.1.1)
  8. cd ~/hexo
  9. hexo init
  10. npm install
  11. npm install hexo-deployer-git –save
  12. 配置_config.yml文件
  13. 下载安装配置NexT主题：教程地址
  14. 添加最近访客功能：教程地址
  15. 域名绑定，添加~/hexo/source/CNAME文件，内容是coderec.cn
  16. 将已有域名配置两条解析『CNAME www coderec.github.io』『CNAME @ coderec.github.io』
  17. 到这里，配置完成，已经可以通过hexo写博客了

由于安装过程网上很多，这里只是记录下流程，搭建容易维护难。希望自己可以坚持写下去。
